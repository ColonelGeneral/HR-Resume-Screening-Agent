## Workflow

1. **Upload Job Description**
   - Enter job title
   - Select job description file
   
2. **Upload Resumes**
   - Select multiple resume files
   - Click upload
   
3. **Start Evaluation**
   - Click evaluate button
   - Wait for AI scoring (4-5 minutes)
   
4. **Review Rankings**
   - See candidates ranked by score
   - View detailed scoring breakdown
   - Override scores if needed

## Troubleshooting

### ❌ Port 3000 already in use
```bash
npm run dev -- --port 3001
```

### ❌ Backend connection error
- Check: Is FastAPI running on port 8000?
- Check: Are CORS headers enabled?
- Check: No firewall blocking localhost?

### ❌ Slow evaluation
- Normal! Embeddings take 4-5 minutes
- Only happens first time per job
